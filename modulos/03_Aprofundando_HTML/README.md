## 🎯 Objetivo
Aprofundar o conhecimento em **HTML**, explorando elementos semânticos avançados, estrutura de documentos, formulários, tabelas, iFrames e novos recursos modernos da linguagem, aplicando tudo na construção de páginas completas.

---

## 📌 Conteúdos Abordados

### 🧩 Elementos Semânticos Avançados
- **Citações** — uso correto de `<blockquote>` e `<cite>`
- **Abreviações** — elemento `<abbr>` com atributo `title`
- **Detalhes de contato** — elemento `<address>`
- **Códigos** — elementos `<code>`, `<pre>` e `<kbd>`
- **Datas** — elemento `<time>` com atributo `datetime`

---

### 🏗️ Estrutura de Documentos
- Aprofundamento na organização semântica de um documento HTML
- Hierarquia de elementos e boas práticas de estruturação

---

### 📋 Formulários
- Criação e estruturação de formulários com `<form>`
- Tipos de `<input>`, `<label>`, `<select>`, `<textarea>` e `<button>`
- Atributos de validação: `required`, `min`, `max`, `placeholder`
- **Input Pattern + Regex** — validação personalizada com expressões regulares

---

### 📊 Tabelas
- Estrutura com `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`
- Atributos `colspan` e `rowspan` para mesclagem de células

---

### 🖼️ Elementos Especiais
- **iFrames** — incorporação de conteúdo externo com `<iframe>`
- **Dialog** — elemento `<dialog>` para modais nativos do HTML
- **selectedcontent** — recurso moderno para manipulação de seleções
