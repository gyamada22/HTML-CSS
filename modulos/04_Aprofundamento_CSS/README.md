## 🎯 Objetivo
Aprofundar o conhecimento em **CSS**, explorando especificidade, espaçamentos, unidades, pseudo-classes, pseudo-elementos, Box Model, posicionamento, backgrounds e novos recursos modernos, aplicando tudo na estilização de páginas completas.
---
## 📌 Conteúdos Abordados
### 🎯 Especificidade e Herança
- **Especificidade no CSS** — como o navegador resolve conflitos entre regras
- **Seletor universal** — `* {}` e sua aplicação no reset padrão
- **Herança de estilos** — quais propriedades são herdadas e como controlá-las
- Reset padrão recomendado:
```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
html {
  font-size: 62.5%;
}
```

---

### 📐 Espaçamentos, Dimensões e Unidades
- **Margin e Padding** — shorthand com múltiplos valores, ex: `margin: 15px 10px`
- **Unidades relativas** — `em`, `rem`, `vh`, `vw`
- **Novas unidades modernas** — `svh`, `lvh`, `dvh` (small, large e dynamic viewport height)

---

### 🎭 Pseudo-classes e Pseudo-elementos
- **Pseudo-classes** — `p:hover {}`, `a:link {}` e a ordem **LVHA**: `link`, `visited`, `hover`, `active`
- **Pseudo-elementos** — `p::first-letter {}`, `p::before {}`, `p::after {}`

---

### 📦 Box Model e Visuais
- **Box Model no CSS** — `content`, `padding`, `border` e `margin`
- **Sombras** — `box-shadow` para elementos
- **Elementos arredondados** — `border-radius {}`

---

### 🖥️ Display e Posicionamento
- **Tipos de visualização** — `display: block`, `inline`, `inline-block`, `none` e suas diferenças
- **Tipos de posicionamento** — `position: static`, `relative`, `absolute`, `fixed`, `sticky`

---

### 🖼️ Background
- **Background Color e Image** — uso de `background-image: url()` e gradientes com `background-image`
- **Controle de exibição** — `background-position`, `background-size: contain / cover`, `background-repeat: no-repeat`
