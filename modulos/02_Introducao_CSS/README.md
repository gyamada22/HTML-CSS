## 🎯 Objetivo
Compreender o funcionamento do **CSS**, aprender as formas de aplicar estilos, utilizar **seletores**, estilizar textos, cores e hiperlinks, e aplicar tudo isso na construção de páginas completas com visual profissional.

---

## 📌 Conteúdos Abordados

### 🧩 Fundamentos do CSS
- Introdução ao CSS e seu papel no desenvolvimento web
- As três formas de escrever CSS:
  - **Inline** — diretamente na tag HTML
  - **Interno** — dentro da tag `<style>` no `<head>`
  - **Externo** — arquivo `.css` vinculado via `<link>`

---

### 🎯 Seletores
- Seletores de **tag**, **classe** (`.classe`) e **ID** (`#id`)
- **Combinando seletores** — seletores descendentes, agrupados e específicos

---

### ✍️ Estilização de Textos
- `font-family` — Família tipográfica (ex: `Montserrat, sans-serif`)
- `font-size` — Tamanho do texto (px)
- `font-weight` — Peso/espessura da fonte (`400`, `600`, `700`)
- `font-style` — Estilo (`normal`, `italic`)
- `text-transform` — Transformação (`uppercase`, `lowercase`)
- `text-decoration` — Decoração (`none`, `underline`)
- `color` — Cor do texto (hex, rgb, nome)

---

### 🎨 Cores
- Utilização de cores em **hexadecimal** (`#112e61`, `#7a40be`, `#f7a70c`)
- Aplicação de cores em textos, fundos e elementos interativos
- Paleta utilizada no exercício:
  - Azul escuro: `#112e61`
  - Roxo: `#7a40be`
  - Azul médio: `#005e88`, `#1a77a4`, `#008ff7`
  - Amarelo: `#f7a70c`
  - Cinza: `#404040`

---

### 🔗 Estilizando Hiperlinks
- Remoção do sublinhado padrão com `text-decoration: none`
- Customização de `color`, `font-weight` e `font-size` em links
- Uso de seletores específicos para links dentro de classes
